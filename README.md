# Munda 🌱

An AI-powered crop advisory and yield prediction platform for Zimbabwean smallholder farmers and extension workers.

## What it does

- **Crop Diagnosis** — farmers describe a problem in plain text and receive instant AI-powered agronomic advice
- **Yield Prediction** — input soil data (NPK, pH, region, crop type) and get a predicted yield with AI analysis
- **Dashboard** — track all past diagnoses and predictions in one place

## Built with

- Next.js 16, TypeScript, Tailwind CSS
- Supabase (Auth + PostgreSQL)
- Drizzle ORM
- Anthropic Claude AI
- Claude Code (agentic coding workflow)

## Why Munda

Munda means "field" in Shona, the language spoken by most Zimbabwean farmers. Extension workers in Zimbabwe are outnumbered — one worker often covers hundreds of farmers. Munda gives every farmer instant access to agronomic expertise without needing a smartphone camera or fast internet.
